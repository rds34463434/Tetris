# Tetris
«Тетрис» — это культовая компьютерная игра-головоломка, изобретённая советским программистом Алексеем Пажитновым в 1984 году, где игроку нужно вращать и перемещать падающие с верха экрана фигурки из четырех квадратов (тетрамино), чтобы составлять из них сплошные горизонтальные линии, которые затем исчезают, принося очки.
